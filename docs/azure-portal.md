# 🖥️ Microsoft Azure Portal

## 📖 O que é o Azure Portal?

O Azure Portal é a interface web da Microsoft Azure para gerenciamento de recursos em nuvem.

Através dele é possível criar, configurar e monitorar serviços.

---

## 🌐 Visão Geral do Portal

### Interface principal do Azure Portal

Aqui está a interface principal utilizada para gerenciar todos os serviços da Microsoft Azure:

![Azure Portal](images/azure-portal.png)

O portal é dividido em áreas principais:
- Dashboard
- Menu lateral de serviços
- Barra de pesquisa
- Resource Groups

---

## 🌐 Acesso

URL oficial:
https://portal.azure.com

---

## 🧭 Principais Componentes

### 📊 Dashboard

Área inicial com visão geral dos recursos.

- Recursos recentes
- Métricas
- Atalhos

---

### ☁️ Resource Groups

Agrupamento lógico de recursos.

**Exemplo:**
- VM
- Banco de dados
- Storage

Tudo dentro de um mesmo projeto.

---

### 🖥️ Virtual Machines

Permite criar servidores virtuais na nuvem.

---

### 📦 Storage Accounts

Armazenamento escalável para arquivos, blobs e dados.

---

### 🗄️ Databases

Serviços como:
- Azure SQL Database
- Cosmos DB

---

### ⚙️ Azure Functions

Execução de código serverless.

---

### 🔐 Identity (Entra ID)

Gerenciamento de usuários e permissões.

---

## 🔧 Funcionalidades Importantes

### 🔎 Search Bar

Permite localizar qualquer recurso rapidamente.

---

### 📌 Pin de Recursos

Fixar serviços no dashboard para acesso rápido.

---

### 📈 Monitoramento

- Logs
- Métricas
- Alertas

---

## 🧠 Boas Práticas

- Organizar recursos em Resource Groups
- Nomear recursos corretamente
- Usar tags para controle financeiro
- Monitorar custos constantemente

---

## 🎯 Resumo

O Azure Portal é o centro de controle da nuvem Azure, permitindo gestão completa de infraestrutura e aplicações.

---

## 🚀 Conclusão

Dominar o portal é essencial para qualquer profissional de Cloud Computing.
